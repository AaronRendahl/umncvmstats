# Scores on pre/post test (generated data)

Generated pre/post data to demonstrate paired tests. A score of 80 is
considered passing.

## Usage

``` r
passfail
```

## Format

A tibble with 50 rows and 4 columns.

- score1:

  pre-test score

- score2:

  post-test score

- pass1:

  pre-test pass/fail status

- pass2:

  post-test pass/fail status
